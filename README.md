# ChatRoom
Chat service built in Go with a microservices architecture.

> ⚠️ **Attention! Project is under development.**  

## What Works?
- Authorization Service
- Room Service

## Planned Features
- Message Service
- Gateway Service

## How to Run
1. Clone the repository
2. Configure `.env`
3. Run `docker-compose up`
